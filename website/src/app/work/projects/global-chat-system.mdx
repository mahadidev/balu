## How It Works

1. **Channel Registration**: Server admins register a channel for global chat
2. **Message Relay**: Messages are instantly broadcast to all connected servers
3. **Moderation**: Automatic filtering and rate limiting protect the network
4. **Logging**: All messages are logged for moderation purposes

## Commands

### Setup Commands
#### New Simplified Commands
- `!chat_subscribe` - Subscribe current channel to global chat network
- `!chat_create <name>` - Create new channel and auto-subscribe to global chat
- `!chat_status` - View network status and connected servers

#### Legacy Commands (still supported)
- `!globalchat register` - Register current channel for global chat
- `!globalchat unregister` - Remove channel from global chat
- `!globalchat status` - View network status and connected servers
